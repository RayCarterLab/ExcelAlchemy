# Security Policy

## Supported Versions

Security fixes are best-effort for the latest released version of ExcelAlchemy.

## Reporting a Vulnerability

Please do not open public GitHub issues for security-sensitive reports.

Use GitHub Security Advisories when possible:

- https://github.com/RayCarterLab/ExcelAlchemy/security/advisories/new

If that workflow is unavailable, contact the maintainers privately and include:

- a clear description of the issue
- affected versions
- reproduction steps or a proof of concept
- any suggested mitigations

We will acknowledge valid reports as quickly as possible, confirm impact, and coordinate a fix before public disclosure when appropriate.
